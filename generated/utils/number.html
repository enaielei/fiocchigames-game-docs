class Number¶

A numerical object that supports clamping, normalization, and dynamic constraints.

This class wraps a numeric value (val) with optional min and max bounds, a normalization interface, and optional value processors.

Parameters:

  • val (float) – The initial value.

  • min (float or None) – Minimum allowed value (inclusive), default is 0.0.

  • max (float or None) – Maximum allowed value (inclusive), default is 1.0.

  • norm (float or None) – A normalized value (between 0 and 1) to initialize from.

  • processor (Callable[[float], float] or None) – A function that transforms all input values (e.g. rounding).

  • use_min (bool) – Whether to enforce the minimum bound.

  • use_max (bool) – Whether to enforce the maximum bound.

max¶
